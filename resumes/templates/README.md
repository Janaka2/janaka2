# Resume Maintenance System

Use this folder to maintain one strong master resume and quickly tailor role-specific versions.

## Files
- `resume_master_template.md`: Canonical long-form source with all relevant history.
- `achievement_bank_template.md`: Reusable impact bullets grouped by domain.
- `tailoring_checklist.md`: 30-minute per-application workflow.
- `role_variant_template.md`: Skeleton for a job-targeted resume.

## Recommended workflow
1. Update `achievement_bank_template.md` weekly with new quantified wins.
2. Refresh your master resume monthly with strongest bullets.
3. For each application, copy `role_variant_template.md`, tailor top 40% content to the job, and keep motivation text job-specific.
4. Store final files with naming convention:
   - `resume_<company>_<role>_<yyyy-mm-dd>.md`
   - `resume_<company>_<role>_<yyyy-mm-dd>.pdf`

## Quality guardrails
- Keep each bullet in the format: **Action + Scope + Tech + Measurable Outcome**.
- Prioritize latest 8–10 years and strongest impact.
- Use job keywords naturally; avoid keyword stuffing.
- Keep private/client-sensitive details generalized.
